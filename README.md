# Vault Visualizer

3D visualizer showing how much physical space a given amount of cash occupies. Built with Three.js, no build step required.

## Features

- 3D vault room with orbit controls (drag to rotate, scroll to zoom)
- AUD denominations: $5, $10, $20, $50, $100
- Stacks cap at 1m height and expand into a grid
- Human scale figure for reference
- Stats bar: note count, stack count, total height, volume, weight

## Banknote images

Note textures use real AUD banknote photographs sourced from the [Reserve Bank of Australia](https://banknotes.rba.gov.au) for educational purposes.

## Usage

Open `index.html` via any HTTP server (e.g. `python3 -m http.server`) or visit the GitHub Pages deployment.
